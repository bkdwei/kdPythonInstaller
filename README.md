# 使用说明
- 首先查看系统信息，如果处理器那里显示x86_64，则安装64位python，否则安装32位python。
- 其次需要安装pip。这个工具用来执行安装命令和卸载命令的。
- 最后，在输入框中输入你要安装的python软件的名称，点击安装即可。
- 如果要卸载python软件，在输入框输入软件名，点击卸载按钮即可。
